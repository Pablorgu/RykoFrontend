# Ryko - Frontend

## Description

This is the mobile frontend for **Ryko**, a comprehensive nutrition management application developed as part of a Bachelor's Thesis project. Built with **React Native** and **Expo**, it offers a seamless and intuitive experience for users to track, manage, and improve their daily nutrition.

## Features

- 🔐 **Secure login** via Google or email and password.
- 🍽️ **Daily meal tracking** with custom dishes and ingredients.
- 📊 **Visual dashboards** showing real-time nutritional data and progress.
- 🎯 **Personalized recommendations** based on user behavior and nutritional goals.
- ⚡️ Fast, responsive, and optimized for mobile devices.

## Stack

- **React Native** with **Expo**
- **TypeScript**
- **React Navigation**
- **REST API integration** with the Ryko backend
